1. What Data We Collect

For X and bluesky no user or follower infomartion is collected.

For Discord MercuryBot only collects server-related data:

  • Server ID (to identify the server)
  • Channel ID (for sending notifications)
  • Role ID (if a role is set to be pinged)
  • Population count (how many users are in the server)
  • Server name
  • Notification settings (selected stores)
  • Date added (when the bot joined the server)

We do NOT collect: Usernames, user IDs, messages, or personal user data.
